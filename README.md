# 📝 Blog Application

A simple and dynamic blog application built using **Node.js**, **Express**, **MongoDB**, and **EJS**.  
This project allows users to create, read, update, and delete blog posts with a clean UI and structured backend.

---

## 🚀 Features

- Create new blog posts  
- View all blog posts  
- Edit existing posts  
- Delete posts  
- Server-side rendering using EJS  
- MongoDB database integration  
- Clean project structure  

---

## 🛠 Tech Stack

- **Backend:** Node.js, Express.js  
- **Frontend:** EJS, HTML, CSS  
- **Database:** MongoDB (Atlas / Local)  
- **ODM:** Mongoose
